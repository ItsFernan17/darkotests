## Instalar Dependencias

```bash
$ npm install
```

## Instalar MySQL en Docker

```bash
$ docker compose up -d
```

## Compilar y Ejecutar el Proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Ejecutar Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
